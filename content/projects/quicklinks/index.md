# QuickLinks

QuickLinks, sık kullandığınız web sitelerine hızlıca erişmenizi sağlayan sade ve işlevsel bir tarayıcı başlangıç sayfasıdır. Tamamen HTML ve CSS kullanılarak inşa edilen bu proje, kullanıcıların bağlantılarını organize etmelerine ve arayüzü kendi zevklerine göre kişiselleştirmelerine olanak tanır.

### Demo

QuickLinks'in canlı demosuna buradan göz atabilirsiniz: [QuickLinks Demo](https://quicklinks-demo.netlify.app).

## Özellikler

- **Hızlı Erişim**: Sık kullanılan web sitelerine tek tıkla ulaşın.
- **Kullanıcı Dostu Arayüz**: Temiz, modern ve sezgisel bir tasarım.
- **Özelleştirilebilir**: Kendi bağlantılarınızı kolayca ekleyebilir ve düzenleyebilirsiniz.

## Kullanılan Teknolojiler

- HTML5
- CSS3 (Custom Variables & Data Attributes)
